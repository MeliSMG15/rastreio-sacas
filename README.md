# Rastreio de Sacas - Stage Zone (SMG15)

App de rastreio de sacas por stage zone. HTML unico + Firebase Realtime Database.

- Live: https://melismg15.github.io/rastreio-sacas/
- Fonte de dados: Apps Script (apps-script-sacas.gs) lendo a aba 'sacas' da planilha.
- Firebase: projeto registrosmg15, no 'registro_sacas'.
